# LookMore Apple Design — visionOS 空间风格前端重设计

> **状态：** 设计已确认，待实施
> **替代：** `2026-05-15-lookmore-ac-style-design.md`（动森露营风格）
> **范围：** 仅前端 UI 层（WXSS + WXML），后端 API 不变，JS 逻辑不变

## 目标

将 LookMore 微信小程序从动森露营手作风格重写为 Apple visionOS 空间风格。使用 iOS 系统蓝 `#007AFF` 作为主色调，通过半透明容器、多层阴影、内发光边框模拟毛玻璃质感，打造空间纵深的视觉体验。

## 技术约束

- **WeChat 小程序不支持 `backdrop-filter`**：毛玻璃效果通过 `rgba` 半透明背景 + `box-shadow` 多层阴影 + `inset` 内发光边框模拟
- **WeChat 小程序不支持 SF Pro 字体**：使用 PingFang SC（Apple 中文系统字体）
- **包体积限制 2MB**：不新增图片资源，仅使用现有 3 张图片
- **实施方案为纯 CSS 重写**：不动 JS 逻辑（流式传输、滚动冻结、文件上传等保持不变）

## 全局设计系统

### 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色调 Primary | `#007AFF` | iOS 系统蓝 |
| 背景 Background | `#F2F2F7` | 浅灰底 |
| 卡片/容器 Surface | `rgba(255,255,255,0.65-0.72)` | 半透明白（根据层级调节） |
| 文字主色 Text Primary | `#1D1D1F` | 近黑 |
| 文字辅助 Text Secondary | `#8E8E93` | 灰 |
| 用户头像边框 | `#FF9500` | iOS 橙色 |
| AI 头像边框 | `#007AFF` | 系统蓝 |

CSS 变量命名：
- `--primary`: `#007AFF`
- `--bg`: `#F2F2F7`
- `--surface`: `rgba(255,255,255,0.68)`
- `--text-primary`: `#1D1D1F`
- `--text-secondary`: `#8E8E93`
- `--shadow-sm`, `--shadow-md`, `--shadow-lg`: 三级阴影
- `--radius-sm`, `--radius-md`, `--radius-lg`, `--radius-full`: 四级圆角

### 圆角

| 级别 | 值 | 用途 |
|------|-----|------|
| sm | `6px` | 小元素（复制按钮、头像） |
| md | `12px` | 按钮、头像、Logo 图标 |
| lg | `20px` | 卡片容器 |
| full | `99px` | 胶囊按钮、顶栏、输入框 |

### 阴影（模拟空间深度）

| 级别 | 值 | 用途 |
|------|-----|------|
| Level 1 | `0 1px 4px rgba(0,0,0,0.04)` | 轻微浮起（气泡） |
| Level 2 | `0 4px 16px rgba(0,0,0,0.08), 0 0 0 0.5px rgba(255,255,255,0.5) inset` | 卡片悬浮 + 内发光 |
| Level 3 | `0 12px 40px rgba(0,0,0,0.12), 0 0 0 1px rgba(255,255,255,0.7) inset, 0 0 30px rgba(0,122,255,0.06)` | 纵深漂浮 + 蓝色光晕 |

### 字体

- 字体族：`PingFang SC, -apple-system, sans-serif`
- 层级：10px（胶囊标签）/ 11px（辅助文字）/ 14px（正文）/ 17px（页面标题）/ 22px（大标题）
- 字重：400 Regular / 500 Medium / 600 Semibold / 700 Bold

## 首页设计

### 空间层次（从后到前）

1. **背景层**：`linear-gradient(180deg, #E8E8ED, #F2F2F7, #FAFAFA)` + 两个蓝色模糊光晕（`border-radius: 50%; filter: blur(60px)` / `blur(50px)`）
2. **Logo 区域**：56px 蓝色圆角方块（12px 圆角）+ 22px Bold "LookMore" + 11px "智能助手"，居中
3. **模型选择胶囊**：页面居中，独立一行。蓝色胶囊 `#007AFF` + 白色文字 + 外发光阴影
4. **主卡片**：92% 宽度居中，22px 圆角，半透明白背景 + 三层阴影 + 内发光边框
5. **Popover 弹出菜单**：点击胶囊后从下方弹出，居中，白色半透明卡片 + 分组模型列表

### 主卡片内部布局

```
┌──────────────────────────────────────┐
│  ┌──────────────────────┐  [↑]  [+] │
│  │  输入你想了解的内容...  │  (蓝)  (灰) │
│  └──────────────────────┘            │
└──────────────────────────────────────┘
```

- 输入框：`flex: 1`，38px 高，14px 圆角，浅灰底
- 发送按钮：34px 蓝色胶囊，↑ 箭头，外发光
- 上传按钮：34px 灰色胶囊，+ 号

### 模型选择 Popover

- 触发：点击模型胶囊
- 位置：胶囊下方，水平居中弹出
- 样式：16px 圆角白卡，多层阴影，最小宽 180px
- 内容：标题「选择模型」+ 基础模型组 + 增强模型组
- 选中项：蓝色高亮 `#007AFF` + ✓
- 关闭：再次点击胶囊 / 点击遮罩 / 选中模型后自动关闭

### 交互流程

1. 默认模型：GPT-5.5
2. 用户输入文字 + 可选上传文件（图片/文件/视频）
3. 点击发送 → `wx.navigateTo` 跳转对话页，携带 model + message + files 参数

## 对话页设计

### 页面结构

```
┌─────────────────────────────────┐
│       [ ← GPT-5.5 ⋯ ]          │  ← 悬浮胶囊顶栏
├─────────────────────────────────┤
│  [AI头像] ┌──────────────┐     │
│           │ AI 半透明白气泡│     │  ← 毛玻璃气泡
│           └──────────────┘     │
│         ┌──────────────┐ [头像]│
│         │ 用户蓝色气泡  │       │  ← 蓝色实色 + 外发光
│         └──────────────┘       │
├─────────────────────────────────┤
│ ┌──────────────────┐ [↑] [+]  │  ← 输入栏
│ │ 输入消息...       │ (蓝) (灰) │
│ └──────────────────┘           │
└─────────────────────────────────┘
```

### 悬浮胶囊顶栏

- 不贴边，居中悬浮
- 半透明白背景 `rgba(255,255,255,0.72)` + 内发光边框
- 99px 圆角胶囊
- 内容：← 返回（蓝色） / 模型名（深色 13px 600） / ⋯ 更多（灰色）
- 间距：左右元素 gap 24px

### 消息行布局

- `.msg-row`：flex 行，`gap: 8px`，`padding: 0 12px`，`margin-bottom: 14px`
- `.msg-row.user`：`justify-content: flex-end`
- 头像：36x36px，8px 圆角，AI 蓝色 / 用户橙色边框（2px）
- 气泡列：`max-width: 65%`，`min-width: 0`
- 气泡列.assistant：`align-items: flex-start`
- 气泡列.user：`align-items: flex-end`

### AI 气泡

- 半透明白背景 `rgba(255,255,255,0.62)`
- 内发光边框 `0 0 0 0.5px rgba(255,255,255,0.5) inset`
- 轻阴影 `0 2px 12px rgba(0,0,0,0.04)`
- 14px 圆角，10px 14px padding，12px 字号
- 文字色 `#1D1D1F`
- 三角箭头：`::before` 伪元素，5px border trick，指向左（AI 头像方向）

### 用户气泡

- 蓝色实色背景 `#007AFF`
- 外发光阴影 `0 4px 16px rgba(0,122,255,0.25)`
- 14px 圆角，10px 14px padding，12px 字号
- 文字色 `#fff`
- 三角箭头：`::before` 伪元素，指向右（用户头像方向）

### 复制按钮

- 18x18px，`opacity: 0.35`，`margin-top: 4px`
- 位于气泡下方，角色对齐（AI 左 / 用户右）
- 仅内容非空时显示

### 流式加载态

- 新消息发送时，立即插入空 assistant 气泡
- 气泡内显示 `...`（灰色文字，通过 `wx:elif` 当 content 为空时显示）
- 流式传输中：每 10ms `setData` 刷新文字，每 200ms `markdownToHtml` 刷新 rich-text
- 流式结束：`loading: false`，气泡内容完整

### 输入栏

- 水平排列：`[输入框 flex:1] [↑发送 蓝色] [+上传 灰色]`
- 输入框：38px 高，19px 圆角（半圆），半透明白背景
- 按钮：34px 胶囊，`flex-shrink: 0`
- 文件 chips：输入栏上方，水平排列

### 滚动行为（保持不变）

- 新消息发送时自动滚到底部（`scroll-into-view="msg-bottom"`）
- 用户上滑 >10px 时冻结自动滚动（`_userScrolledUp = true`）
- 发送新消息时重置冻结状态
- `bindscroll="onScroll"` 监听滚动位置

## 设计决策记录

| 决策点 | 选择 | 备选方案 |
|--------|------|----------|
| 视觉风格 | visionOS 空间风格 | iOS 现代 / macOS / 极致极简 |
| 主色调 | iOS 蓝 `#007AFF` | 深空灰 / 自定义 |
| 容器质感 | 半透明白 + 多层阴影模拟毛玻璃 | — |
| 首页布局 | 纵深分层（光晕 + 卡片 + Popover） | 居中卡片 / 全屏 Hero |
| 聊天气泡 | 毛玻璃气泡（AI 半透白 / 用户蓝实色） | 全半透明统一 / 无气泡极简 |
| 模型选择 | Popover 弹出菜单，居中 | 底部 Sheet / 内联胶囊组 |
| 对话顶栏 | 悬浮胶囊（不贴边） | 无缝沉浸贴边 |
| 按钮顺序 | 发送 ↑ 在左，上传 + 在右 | — |
| 实施方案 | 纯 CSS 重写，不动 JS | 深度重构 / 渐进式升级 |
