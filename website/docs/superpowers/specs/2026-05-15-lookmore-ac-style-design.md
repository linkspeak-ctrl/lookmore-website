# LookMore 动森风格前端重设计 — 设计规范

> **目标**：将 LookMore 微信小程序前端重新设计为 Animal Crossing（动物森友会）露营手作风格，2 个页面（首页 + 对话页）。

**设计基调**：露营手作 — 亚麻质感、鼠尾草绿点缀、陶土色辅助、深木色文字。圆角、柔软、治愈。

---

## 1. 全局样式

### 1.1 配色

| 角色 | 色值 | 用途 |
|------|------|------|
| 页面底色 | `#EFEBE9` | 全局背景 |
| 卡片/气泡白 | `#FFFFFF` | 输入框、消息气泡 |
| 鼠尾草绿（主强调） | `#A5D6A7` | 按钮、描边、标签、用户气泡 |
| 深鼠尾草绿 | `#81C784` | 按钮按下态 |
| 陶土色（辅助） | `#BCAAA4` | 次要文字、分隔线 |
| 淡奶油 | `#FFF9C4` | 高亮/强调背景 |
| 深木色（主文字） | `#4E342E` | 标题、正文 |
| 浅木色（次文字） | `#8D6E63` | 副标题、时间戳 |

### 1.2 字体

- 系统默认字体（`-apple-system, BlinkMacSystemFont, sans-serif`）
- 品牌名 "LookMore" 使用 Caveat 手写字体（36px bold），飘逸轻快
- 正文 11-14px

### 1.3 全局圆角

- 大卡片/容器：16-20px
- 按钮/标签：12-16px  
- 气泡：14px，靠近头像一侧为 4px 直角

### 1.4 装饰元素（C 方案 — 融入组件）

动森元素直接点缀在 UI 组件中，而非背景层。所有装饰图标仅作氛围点缀，不参与交互。

## 2. 页面一：首页

### 2.1 布局结构

```
┌────────────────────┐
│                    │
│    [小程序头像]     │  ← 72px 圆形，3px #A5D6A7 描边
│    LookMore        │  ← 22px bold #4E342E
│    撸猫 AI 助手     │  ← 11px #8D6E63
│                    │
│  [    对话框    ]   │  ← 260px 宽，与 Logo 对齐居中
│  🍃 GPT-5.5 ▾       │    模型选择在对话框下方居中
│                    │    品牌名 LookMore 使用 Caveat 手写体 36px
│                    │
└────────────────────┘
```

### 2.2 模型选择（对话框下方居中）

- 位于对话框下方 14px，单独居中
- 鼠尾草绿 `#A5D6A7` 圆角标签，padding 6px 12px，border-radius 12px
- 文字前 🍃 叶子图标，文字 `GPT-5.5 ▾`，11px bold `#33691E`
- 点击展开下拉菜单（在按钮下方弹出）

### 2.3 下拉菜单

- 白色圆角卡片（border-radius 14px），带阴影
- 分"基础模型"和"增强模型"两组，每组有小标题
- 当前选中项高亮（鼠尾草绿色背景）
- 点击选项→更新按钮文字+关闭菜单
- 点菜单外区域→关闭菜单

模型列表（最终版本来自 server/lazymanchat.js MODELS 数组）：

**增强模型**：
- GPT-5.5 (OpenAI)

### 2.4 对话框（输入区）

- 宽度 260px，与上方 Logo 区块对齐居中
- 白色背景卡片，18px 圆角，2px `#C8E6C9` 描边
- 左上角 🪵 木材装饰（16px，绝对定位），右下角 🏕️ 帐篷装饰（14px，绝对定位）
- 内部：输入框 + 发送按钮，水平排列
- 输入框 placeholder "输入你想问的问题..."
- 发送按钮：26px 鼠尾草绿 `#A5D6A7` 圆形，白色 ↑ 箭头

### 2.5 交互

- 点击发送按钮或键盘确认 → 跳转到对话页（携带选中的模型 ID 和消息内容）
- 首次进入默认模型 GPT-5.5
- 首页无提示文字

## 3. 页面二：对话页

### 3.1 布局结构

```
┌────────────────────┐
│ ←  GPT-5.5         │  ← 顶部栏 44px，底部分隔线
├────────────────────┤
│                    │
│ [🐱] LookMore助手   │  ← AI 消息：左头像 + 名称 + 白气泡
```

## 5. 技术要点

- 平台：微信小程序（WXML + WXSS + JS）
- 全局样式写在 `app.wxss`，页面特有样式写在各页面 wxss
- 头像图片需要复制到 miniprogram 资源目录中（如 `miniprogram/images/`）
- 后端 API 保持不变（`server/routes.js`），前端只需调整 `utils/api.js` 的调用参数
- 需要处理模型列表的硬编码（前端应有一份模型列表，用于下拉菜单渲染）

## 7. 设计稿参考（动森露营风格关键字）

- 底色亚麻灰 `#EFEBE9`，让人联想到动森中铺装的小路和帐篷帆布
- 鼠尾草绿 `#A5D6A7` 是动森中树叶和新芽的颜色
- 陶土色 `#BCAAA4` 像动森的花盆和陶器
- 深木色 `#4E342E` 是树木和木质家具的颜色
- 圆角大、线条柔软，对应动森的 Q 版画风
- 组件装饰图标（🍃🪵🏕️🧵🌱🌳）点缀在卡片、按钮、标签中，强化露营手作氛围
